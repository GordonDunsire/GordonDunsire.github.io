Map from RDA properties to MARC Code List for Relators

This map relates RDA relationship designators to codes from the MARC list for relators.

The map is given in a terse triple language (ttl) serialization.

The subject of each mapping is an RDA unconstrained property with no domain or range and typed as rdfs:Property. The property represents a relationship designator from Appendix I of the RDA Toolkit, not differentiated with respect to RDA entities.

The predicate of each mapping is a SKOS property with no domain or range.

The object of each mapping is the URI of a MARC relator code with no domain or range and typed as skos:Concept, rdfs:Property, owl:ObjectProperty, and mads:Topic.

This map is embedded in the Unconstrained properties element set.
