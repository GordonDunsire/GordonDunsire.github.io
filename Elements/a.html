RDA element sets

Agent properties

The Agent properties element set consists of properties representing the attributes and relationships of the RDA Agent, Person, Family, and Corporate Body entities. Each property in the element set has a domain of the class representing one of these entities.

Number of elements: 226
Namespace: http://rdaregistry.info/Elements/a/
Suggested prefix: rdaa
Example curie (canonical)*: rdaa:P50026
Example curie (lexical)*: rdaa:employee
Changelog feed: Atom | RDF

*All RDA URIs have both an immutable canonical form and a 'readable', lexical form, which is subject to change (changes will be redirected).
